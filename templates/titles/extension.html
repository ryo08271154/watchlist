視聴記録拡張機能について

この拡張機能を使用することで、動画配信サービスから簡単に作品やエピソードのレビューを記録できます。

{% if download_url %} Chrome拡張機能をダウンロード {% else %} 拡張機能は現在利用できません。 {% endif %}

インストール手順

  1. 上記のリンクから最新リリースをダウンロードし、解凍します
  2. Chrome拡張機能の管理ページ(chrome://extensions/)を開きます
  3. 右上の「デベロッパーモード」を有効にします
  4. 「パッケージ化されていない拡張機能を読み込む」をクリックし、解凍したディレクトリを選択します
  5. 拡張機能の設定から視聴記録のURLを {{ request.scheme }}://{{ request.get_host }} にします

使い方

作品・エピソードの確認

  1. 動画配信サービスで作品を開くと、自動的に視聴記録から作品とエピソードを検索します
  2. 拡張機能のアイコンをクリックすると、検出した作品とエピソードの情報が表示されます
  3. 作品名やエピソード名をクリックすると、該当ページが開きます

レビューの記録

フローティングボタンから記録する方法

  1. 拡張機能の設定で「エピソードのレビューボタンを表示する」を有効にします
  2. 動画再生ページで動画を一時停止または再生終了すると、レビューボタンが表示されます
  3. ボタンをクリックすると、エピソードのレビューページが開きます
  4. 右クリックで、エピソードの詳細ページを開けます

拡張機能のポップアップから記録する方法

  1. 動画配信サービスで作品を開いた状態で、拡張機能のアイコンをクリックします
  2. 検出された作品とエピソードの情報が表示されます
  3. 「タイトルをレビュー」をクリックすると作品のレビューページが開きます
  4. 「エピソードをレビュー」をクリックすると現在のエピソードのレビューページが開きます
  5. 「エピソード一覧」をクリックすると作品の全エピソード一覧が開きます

主な機能

  • 視聴中の作品を視聴記録から自動検索
  • エピソードの自動検出
  • ポップアップから作品・エピソードページにアクセス
  • 動画配信サイトからワンクリックでレビューを記録
  • キーワードから視聴記録を検索

注意

  • 動画配信サイトの仕様変更により、正常に動作しなくなる可能性があります。
  • 拡張機能はベータ版であり、予期せぬ動作をする可能性があります。
